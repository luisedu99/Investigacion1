Que es la etiqueta Canvas

El elemento de canvas HTML 5 es un modelo de procedimiento de bajo nivel que se utiliza para crear una representación dinámica y programable de formas 2D e imágenes de mapa de bits.

HTML 5 es un área de mapa de bits de la pantalla que funciona en un modo inmediato, donde el modo inmediato se refiere al orden en que el lienzo representa los píxeles en la pantalla. La pantalla de mapa de bits de cada fotograma se actualiza con llamadas a la API de Canvas desde JavaScript.

Un programador puede dibujar diferentes formas, texto y mostrar imágenes directamente en un área definida de la ventana del navegador usando HTML 5 Canvas API que incluye contexto 2D.


Compatibilidad con el navegador HTML5 Canvas

Aparte de IE 8, casi todos los navegadores web son compatibles con HTML5 Canvas. Los navegadores que admiten HTML 5 Canvas son: Google Chrome, seguido de cerca por Safari, Firefox y Opera. Además, Internet Explorer, versión 9 promete tener soporte para Canvas. El modelo de objetos de documento (DOM) y los objetos de lienzo en una página HTML están representados por el modelo de objetos de documento (DOM). DOM es un lenguaje de plataforma neutral accesible a través de Javascript. Permite que el estilo y el contenido de la página se actualicen después de que se muestre en el navegador web. Sin embargo, utilizando DOM, el elemento Canvas es accesible; DOM no puede acceder a los elementos gráficos creados en el lienzo. La razón detrás de esto es que Canvas funciona en el modo Inmediato.
